## 🎯 Hızlı Başlangıç

### Tasarım Sistemi Özeti

**Renkler:**
- Primary: `#111827` (Antrasit)
- Accent: `#F59E0B` (Amber)
- Background: `#FFFFFF` (Beyaz)

**Typography:**
- Font: Inter
- Heading: Bold/SemiBold
- Body: Regular/Medium

**Breakpoints:**
- Mobile: 0-640px
- Tablet: 640-1024px
- Desktop: 1024px+

## 🛠️ Teknoloji Stack

- **React 18** + **TypeScript**
- **Vite** (Build tool)
- **Tailwind CSS** (Styling)
- **Shadcn UI** (Components)
- **Zustand** (State management)
- **React Router** (Routing)
- **Framer Motion** (Animations)

## 📝 Notlar

- Tüm tasarımlar **mobil öncelikli** (Mobile First)

- **SEO** dostu yapı
